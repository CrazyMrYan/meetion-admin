<h1>meetion-admin</h1>
<h2>install</h2>
<p>meetion-admin 是一款 与 <b>meetion-ui</b> 相结合的单页后台管理系统 生态系统有 <b>router request meetion</b> </p>
<p>Meetion-Admin配合Meetion-Ui可以获得极高的UI定制性，当你需要开发拥有独特风格的WEB站点时，这将是你的不二之选；借助丰富的JS插件， Meetion-Admin也可以满足中自定义开发需求</p>
